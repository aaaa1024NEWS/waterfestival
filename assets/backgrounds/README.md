# 게임 배경 이미지 폴더

현재 게임에서 사용 중인 기본 배경 파일은 아래 파일입니다.

```text
assets/game-background.png
```

새 배경을 여러 개로 나누어 관리하고 싶다면 이 폴더에 넣어주세요.

## 권장 파일 이름

```text
stage-1-background.png
stage-2-background.png
stage-3-background.png
```

## 현재 상태

현재는 별도 스테이지 배경 폴더 대신 `assets/game-background.png` 하나를 게임 배경으로 사용하고 있습니다.

스테이지별 배경 이미지를 넣은 뒤 이렇게 알려주세요.

```text
assets/backgrounds 폴더에 스테이지별 배경 이미지를 넣었어. 게임 배경에 적용해줘.
```
