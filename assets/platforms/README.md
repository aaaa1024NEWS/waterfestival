# 발판 요소 이미지 폴더

게임 속 발판, 바닥, 벽타기 벽, 위험 지형에 사용할 이미지를 이 폴더에 넣어주세요.

권장 형식은 투명 배경 `png`입니다. 도트게임 느낌을 유지하려면 픽셀아트 스타일 이미지가 가장 잘 어울립니다.

## 권장 파일 이름

아래 이름으로 넣어주면 제가 코드에 바로 연결하기 쉽습니다.

```text
platform-ground.png
platform-rock.png
platform-wall.png
platform-waterfall-wall.png
hazard-water.png
hazard-swamp.png
goal-gate.png
```

## 용도

```text
platform-ground.png          넓은 바닥 발판
platform-rock.png            일반 바위 발판
platform-wall.png            수직 벽/기둥
platform-waterfall-wall.png  벽점프용 폭포 벽
hazard-water.png             물/급류 위험 구간
hazard-swamp.png             수렁 위험 구간
goal-gate.png                다음 스테이지로 가는 아치 게이트
```

이미지를 넣은 뒤 이렇게 알려주세요.

```text
assets/platforms 폴더에 발판 이미지를 넣었어. 게임 발판에 적용해줘.
```
