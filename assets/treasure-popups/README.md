# 보물 획득 설명 이미지 폴더

보물을 획득했을 때 화면에 띄울 설명 카드 이미지를 이 폴더에 넣어주세요.

예상 연출:

1. 보물을 획득한다.
2. 보물 이미지 또는 설명 카드가 화면에 나타난다.
3. 보물 이름과 짧은 설명이 함께 표시된다.

권장 형식은 `png`입니다. 설명 카드 전체를 이미지로 만들 수도 있고, 보물 대표 이미지만 넣고 설명 문구는 코드로 표시할 수도 있습니다.

## 권장 파일 이름

보물 이미지와 같은 순서로 맞추면 관리가 쉽습니다.

```text
01-popup.png
02-popup.png
03-popup.png
...
27-popup.png
```

또는 보물 이름 기반으로 넣어도 됩니다.

```text
popup-표고버섯.png
popup-청태전.png
popup-우드랜드.png
```

이미지를 넣은 뒤 이렇게 알려주세요.

```text
assets/treasure-popups 폴더에 보물 설명 이미지를 넣었어. 획득 효과에 연결해줘.
```
